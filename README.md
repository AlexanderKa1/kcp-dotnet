# kcp-unity
A KCP C# implementation for Unity3D

**CAUTION** this project is still NOT production ready.

## Prerequisites

* .net core 2.0 (https://dotnet.github.io/)
* Golang (http://golang.org)

## How to 

### Start Server

```
go run server.go
```

### Start Client
```
git clone https://github.com/ichenq/kcp-unity
cd kcp-unity
dotnet run
```
